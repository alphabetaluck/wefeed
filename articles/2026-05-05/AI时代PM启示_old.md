这篇文章通过 Anthropic 的 Claude Code 产品负责人 Cat Wu 的分享，讨论了 AI 时代产品经理角色的变化：PM 不再主要承担协调和排期，而是更需要判断方向、定义成功标准，并在模型能力快速变化时持续做取舍。

文章强调，AI 公司竞争的不只是模型能力，更是组织速度与发布机制。Anthropic 的做法是把功能尽量快速推向用户，在 research preview 中收集反馈，再靠轻量流程快速迭代。PRD 仍然有用，但更重要的是让团队清楚目标、约束和成功标准。

另一个重点是角色边界被打碎：PM、工程师、设计师都在互相渗透，真正稀缺的能力变成了产品品味、第一性原理思考和主动补位。Cat 认为，最难的不是“AI 能做什么”，而是让 AI 在真实场景里足够可靠；因此评估体系、反馈闭环和持续修正非常关键。

文中也区分了 Anthropic 自家工具的分工：Claude Code 更适合代码与测试，CoWork 更适合文档、PPT、邮件和会议材料。整体来看，这篇文章传达的核心结论是：AI 提升了杠杆，但真正拉动杠杆的人，仍然是有判断力、行动力和责任感的人。
